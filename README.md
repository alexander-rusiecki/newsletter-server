# Simple application to sign up for a newsletter (backend)

## Tech stack

- Express
- Mongoose
- Jsonwebtoken
- Ejs
- Bcrypt
- Express-rate-limit
- Xss-clean

### Start project by:

- creating .env file (read .env.example for required environment variables)
- npm install
- npm start (node)
- npm run dev (nodemon)

[Live demo](https://newsletter-app-server.herokuapp.com/)
